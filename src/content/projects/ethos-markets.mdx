Ethos.markets is a reputation market that allows people to trade on others' credibility. It is similar to Polymarket, but there is no ending period in Ethos.markets. My job was to create UI pages using Remix.js and TailwindCSS and integrate the front end with the contract manager. I also helped create back-end APIs using Node.js, Express.js, and PostgreSQL.

## Key Features

- Reputation trading marketplace
- Continuous market with no ending period
- Smart contract integration for trading
- Real-time credibility scoring
- Advanced trading interface

## Key Contributions

- Created UI pages using Remix.js and TailwindCSS
- Integrated frontend with smart contract manager
- Built back-end APIs using Node.js and Express.js
- Implemented PostgreSQL database integration
- Developed real-time trading functionality
