## Overview

Worked with Air Labs Inc through Toptal as a Front-end React Engineer to ship new features for their Creative Ops platform for visual assets and enhance their marketing website.

## Key Achievements

• **Creative Ops Platform Development**: Implemented new features for the visual assets platform using Next.js, Redux, and React Query. Built a system similar to Google Drive with flexible organization and collaboration features

• **Marketing Website Revamp**: Revamped marketing and blog pages using Next.js, Plasmic, and Contentful for modern content management

• **Bug Resolution & Quality Assurance**: Tracked and resolved platform issues using Jira, ensuring stability and optimal user experience

## Technologies Used

- **Frontend**: React, Next.js, TypeScript, Redux
- **Styling**: Tailwind CSS, Styled-components, CSS
- **State Management**: Redux, React Query
- **CMS**: Contentful, Plasmic
- **Tools**: Webpack, Figma, GitHub, Jest
- **Cloud**: AWS (Amplify, S3), Vercel
- **APIs**: REST APIs, Stripe integration
