## Overview

Worked as a React Developer at Mosanada, building the frontend of their internal dashboard for managing chatbots and integrating chatbot services across multiple social messaging platforms.

• **GraphQL API Integration**: Implemented GraphQL APIs using Apollo and higher-order components for efficient data management and real-time updates

• **Component Architecture**: Created reusable React components with responsive design for optimal dashboard accessibility and user experience
