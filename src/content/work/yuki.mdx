## Overview

Worked with Yuki LLC through Toptal as a Front-end React Engineer to revamp their marketing website and enhance their Web3 gaming platform GoL2.io with modern design and performance optimization.

## Key Achievements

• **Marketing Website Revamp**: Led complete redesign of Yuki website using Next.js, Tailwind CSS, and GSAP animations. Created project from scratch following Figma designs and best practices

• **GoL2.io Gaming Platform Enhancement**: Added NFT minting features for game snapshots and optimized performance through infinite scrolling and pagination improvements

• **Custom Video Player Development**: Built custom video player with Video.js controls and Cloudflare Stream integration for enhanced media experience

• **Animation & UI Development**: Created custom Lottie animation player and reusable UI components for improved visual experience

## Technologies Used

- **Frontend**: React, Next.js, TypeScript, Gatsby
- **Styling**: Tailwind CSS, SCSS, Sass
- **Animations**: GSAP, Lottie for React
- **Web3**: Starknet.js, Web3.js, MetaMask
- **State Management**: Zustand, React Query
- **UI Components**: Radix UI, custom components
- **Tools**: Vite, Vercel, Storybook
